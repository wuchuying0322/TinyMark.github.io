* 作用域：
    ~~~js
    1. js中的作用域，也叫词法作用域

    2. js中只有函数能够形成一个作用域，所以，又叫数作用域

    3. 变量的作用范围在书写代码的时候，就定下来了与代码怎么调用无关

    4. 对于函数来说，创建好了函数以后，函数的作用就定下来了，也就是作用域链也就定下来了

    5. 也就是说不管在哪个位置调用的函数，都是会回创建它的环境中去执行！
    ~~~

* 闭包
    ~~~js
    1. 函数内部声明的变量，在函数外部无法访问，这时候就形成了一个封闭的空间，也就是闭包

    2. 闭包包括了当前函数的函数体和函数所处的环（作用域）

    3. 闭包其实也就是对作用域的应用

    4. 闭包的模型，也就是函数内部嵌套函数
        var fn = function(){
            var num = 100;
            // 闭包的质就是返回一个函数，这个函数携带了函内部的变量，以达到让函数外部也可以访到函数内部声明的变量
            return function(){		
            // 该匿名数是在fn函数的作用域内创建的，所以该数可以访问到fn函数内部声明的变量
                console.log(num);
            }
        }
        console.log(num);			
        // fn函数面访问不到fn函数内部声明的变量
        > undefined
        fn()(); 					
        // 这里函返回值是一个函数，所以还有进行一次调用
        > 100 
        
    5. 闭包的好处：可以包含函数内部私有状态，也就保护函数内部变量
    ~~~
